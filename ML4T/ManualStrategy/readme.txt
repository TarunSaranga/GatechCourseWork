#README

The included files are:
1) indicators.py
2) marketsimcode.py
3) ManualStrategy.py
4) TheoreticallyOptimalStrategy.py

Part 1:
To get charts of indicators run indicators.py by calling main function.

Part 2:
To get charts for optimal strategy run TheoreticallyOptimalStrategy.py by calling the main function. (uses marketsimcode.py)
testPolicy function API is implemented as required.

Part 3:
To get charts for manual strategy run ManualStrategy.py by calling the main function. (uses marketsimcode.py and indicators.py)
testPolicy function API is implemented as required.
